Quick start

One free API key is all you need. Get a Groq key (free, no credit card) and you're running in 60 seconds.

1
Add to your MCP config
{
  "mcpServers": {
    "refactory": {
      "command": "npx",
      "args": ["@refactory/mcp"],
      "env": { "GROQ_API_KEY": "your-key-here" }
    }
  }
}
2
Tell your AI tool to decompose a file
> Analyze and decompose src/big-file.js into modules
3
Or use the CLI directly
$ npx @refactory/mcp decompose src/big-file.js

No API key yet? Run npm run demo after cloning — it decomposes a sample file locally with no keys required.

Hybrid pipeline

AI plans the boundaries. Code does the extraction. One command.

1 ANALYZE Scan file health, function map, dependencies — mechanical
2 CHARACTERIZE Snapshot exports before touching anything — mechanical
3 PLAN LLM decides module boundaries from function map — the only AI step
4 EXTRACT Mechanical: copy functions by line range, resolve imports — zero tokens, instant, 100% valid. LLM fallback for unsupported languages.
5 FIX-IMPORTS Rewrite require()/import paths — mechanical
6 VERIFY Syntax check, load check, export comparison — mechanical
7 METRICS Refactory Score + HTML report with diagrams — mechanical

Refactory Score

One number that tells you if the decomposition worked.

1.0

Every extracted module loads cleanly and the largest module is smaller than the original file. Anything below 1.0 tells you exactly what degraded.

score = clean_rate × size_reduction
clean_rate — modules that load without errors / total modules size_reduction — 1 − (largest output module / original file)
